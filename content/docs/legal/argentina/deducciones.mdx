description: Jubilacion 11% + PAMI 3% + Obra Social 3% = 17%.

## Base legal

- Ley 24.241 (SIPA), Ley 19.032 (PAMI), Ley 23.660 (Obras Sociales).

## Formula

- `jubilacion = base * 0.11`
- `pami = base * 0.03`
- `obra_social = base * 0.03`
- `total = base * 0.17`
